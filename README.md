## Literární období — pokrytí

Podle sylabu Literatura pro 2. stupeň ZŠ:

1. Starověké písemnictví (mimoevropské, antika)
2. Středověká literatura (staroslověnština, husitství)
3. Renesance a humanismus (Dante, Shakespeare)
4. Baroko (Komenský)
5. Klasicismus a osvícenství (Molière, Voltaire)
6. Národní obrození (Mácha, Erben, Němcová)
7. Romantismus (Mácha, Hugo, Byron)
8. Realismus (Tolstoj, Dickens, Jirásek)
9. Česká literatura 2. pol. 19. století (májovci, ruchovci, lumírovci)
10. Přelom 19. a 20. století (Březina, Bezruč)
11. Literatura 1. pol. 20. století (Čapek, Hašek, Kafka)
12. Literatura 2. pol. 20. století (Hrabal, Kundera, Havel)
13. Současná literatura (Viewegh, Tolkien, Murakami)

## Funkce

- **Flashcards** + odpověď + vysvětlení
- **Leitner systém** (5 boxů, intervaly 1/3/7/14/30 dní)
- **Kvízy** s vysvětlením
- **Streak** (dny v řadě)
- **localStorage** ukládání (každá oblast má svůj klíč)
- **Zapínání/vypínání témat** — žák si vybírá
