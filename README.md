# Lingua · Český jazyk pro 2. stupeň ZŠ

Studijní webová aplikace pro 6.–9. ročník ZŠ Lingua Universal.

## Soubory

```
index.html              ← Landing page (3 oblasti ČJL)
pravopis.html           ← Pravopis (modrá)
slohove-utvary.html     ← Slohové útvary (oranžová)
literarni-druhy.html    ← Literární druhy a žánry (tmavá)
data-pravopis.js        ← Karty pro pravopis
data-sloh.js            ← Karty pro slohové útvary
data-literatura.js      ← Karty pro literární druhy
logo.png                ← Školní logo
```

## Funkce

- **Flashcards** + odpověď + vysvětlení
- **Leitner systém** (5 boxů, intervaly 1/3/7/14/30 dní)
- **Kvízy** s vysvětlením
- **Streak** (dny v řadě)
- **localStorage** ukládání (každá oblast má svůj klíč)
- **Zapínání/vypínání témat** — žák si vybírá
